Shell Console
==============

User console is a command line tool useful to manage Nivola platform. It expose command to configure and control
cmp user assets.

To run cli use the command **beehive** with a working section. An example of help that you can obtain is the following:

.. code-block:: bash

    $ beehive -h

      usage: beehive (sub-commands ...) [options ...] {arguments ...}

      Beehive manager.

      commands:

        business             Business Service and Authority Management
        ...

      optional arguments:
        -h, --help            show this help message and exit
        ...

Cli can be used in interactive mode (not completely tested. There are still some bugs) or in simple command mode.

**Simple command mode**:

.. code-block:: bash

    $ beehive env list
    name         current    is_default    has_cmp    openstack    vsphere    veeam           graphite    zabbix
    -----------  ---------  ------------  ---------  -----------  ---------  --------------  ----------  --------
    podto1       False      False         True       podto1       podto1     podto1,tstsddc              podto1
    .........
    test         False      False         True       test         test

**Interactive mode**:

.. code-block:: json

    $ beehive (sub-commands ...) [options ...] {arguments ...}
    beehive> env list
    name         current    is_default    has_cmp    openstack    vsphere    veeam           graphite    zabbix
    -----------  ---------  ------------  ---------  -----------  ---------  --------------  ----------  --------
    podto1       False      False         True       podto1       podto1     podto1,tstsddc              podto1
    .........
    test         False      False         True       test         test                                   test
    beehive>
