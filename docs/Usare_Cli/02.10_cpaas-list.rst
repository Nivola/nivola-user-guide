Ottenere l'elenco delle Virtual Machine.
========================================

Per ottenere l'elenco, delle Virtual Machine, il comando è da usare in questo modo:

.. code-block:: bash

    [73300@cmpto1-cons02 ~]$  beehive ssh nodes get <-size 0>
